# 📋 Project Coordinator

You lead the team through the hour. Drop your artefacts here.

## What lives in this folder
- `team-plan.md` — the agreed site structure (your **first job**, in the opening minutes)
- `status-updates.md` — short Slack-style updates as the hour progresses
- `blockers.md` — anything blocking a teammate that the Product Owner should know about
- `handoff-checklist.md` — your final 5-min handoff to the Coding Agent

## Templates

### `team-plan.md`
```
# Site structure (agreed at minute 7)
- Pages: <list>
- Each page contains: <bullets>
- Hero image direction: <one line>
- Tone: <one line>
- Call to action: <one line>
```
