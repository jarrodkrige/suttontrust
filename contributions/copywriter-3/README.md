# ✍️ Copywriter 3

You're the third copywriter on the team. CW1 owns the hero (headline + lede + CTA). CW2 owns the body sections (what / who / how / buddy). **Your job is voice consistency** — read what CW1 and CW2 commit, then submit alternative drafts and polish so the whole page sounds like it came from one warm, honest voice.

Your work *competes* with CW1's and CW2's drafts. The Factory Agent picks the strongest version per section.

## What lives in this folder
- `voice-rules.md` — the tone rules you'd defend (e.g. *"never say 'empower'"*, *"contractions are fine"*, *"reading age 14"*)
- `alternative-drafts.md` — your re-takes on any section CW1 or CW2 has drafted
- `polish-suggestions.md` — small line-by-line edits you'd push through

## Coordinate with
- **Copywriter 1** — read their headlines/CTA first; offer alternatives if you spot a tonal slip
- **Copywriter 2** — same for body copy
- **Researcher** — make sure stats are woven in naturally
- **Accessibility Champion** — they review for plain language

## First 2 minutes
1. Sign in to copilot.microsoft.com with your role-card account.
2. Read what CW1 and CW2 have dropped (or chat to them).
3. Ask:
   > "Here's the headline our team drafted: '<paste>'. Give me 3 alternative versions in a warmer, more honest voice — for 16-18 year-old UK first-gen students. What's the strongest?"
4. Drop the best one in `alternative-drafts.md`.

## Prompts that work
- *"Read these two paragraphs side-by-side. Where does the voice break? Suggest a single edit to each that fixes it."*
- *"Rewrite this without any abstract nouns ('opportunity', 'potential', 'journey'). Concrete only."*
- *"Score this for plausibility: does it sound like a real person wrote it for a real friend?"*

## House rule
**You're a polish round, not a rewrite round.** Don't bin CW1 or CW2's work — sharpen it.
